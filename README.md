# Eventory

"Eventory" is an innovative Centralized Club Event Advertisement App designed to 
transform the way campus communities manage and promote club events. This comprehensive 
platform offers a range of essential features, including club registration and profile creation,
event publication and approval, efficient event registration with online payment processing, 
timely event notifications, attendance tracking, merit awarding, event invitations sharing, and 
user dashboards. The app's primary goal is to centralize event-related activities within campus 
communities, enhancing communication, engagement, and overall campus life.
